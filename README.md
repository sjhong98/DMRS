## 1. 프로젝트 개요

DMRS(DID Medical Record System)은 블록체인 기반 신원정보 체계 DID를 활용한 **진료기록 관리 시스템**입니다. <br />
환자는 원하는 범위 내에서 자신의 진료기록을 의사에게 제공할 수 있으며, 이를 통해 의사는 진료기록을 열람 및 수정할 수 있습니다.
